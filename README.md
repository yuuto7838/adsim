# Ad Operations Simulator (広告運用シミュレーター)

AIクライアント相手に広告運用を行うシミュレーションゲームです。
Gemini APIを利用して、毎回異なる商材・クライアントが登場します。

## 遊び方
1. ゲームを開始すると、Gemini APIキーの入力が求められます (Google AI Studioで取得可能)。
2. クライアントからのブリーフ(依頼)を読み、プランニングを開始します。
3. 予算を配分し、配信を実行してください。
4. 結果に応じてクライアントの反応が変わります。3ヶ月に1度は「定例報告会」が発生します！

## 技術スタック
- HTML5 / CSS3 / Vanilla JavaScript
- Google Gemini API (1.5 Flash / 2.5 Flash)
